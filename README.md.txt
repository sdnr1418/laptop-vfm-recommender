# 💻 Laptop VFM Recommender

This Streamlit app helps users find the top 10 laptops offering the best **Value for Money (VFM)** based on:
- CPU
- RAM
- GPU
- Storage
- Price

### ⚙️ Features:
- Enter your budget range 💵
- See full specs for top laptops in that range 🖥️
- Click link to search for the exact laptop on Amazon or Google 🛍️
